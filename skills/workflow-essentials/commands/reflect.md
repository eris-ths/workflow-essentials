# /reflect — Learning Record

今回のセッションの学びを振り返り、MEMORY.md に記録する。
次の自分が同じ罠を踏まないために。

**使い方**: `/reflect [--clean]`

## 引数

- **引数なし** → セッションの学びを記録
- **--clean** → MEMORY.md の古い・重複した記録を整理

## 手順

### /reflect

1. セッションの振り返り:
   - 変更した内容を確認
   - エラーや問題の解決過程を振り返る
   - 重要な判断とその理由を整理

2. 記録すべき学びを抽出:
   - 新しいパターン（次も使える手順やコマンド）
   - 踏んだ罠（原因 + 回避策）
   - 重要な判断（選択肢 + 理由 + 結果）

3. MEMORY.md に構造化して追記:

```markdown
## YYYY-MM-DD: [タスク名]
- **What**: 何をしたか
- **Learned**: 何を学んだか
- **Trap**: 踏んだ罠（あれば）
- **Decision**: 重要な判断（あれば）
```

### /reflect --clean

1. MEMORY.md を読む
2. 古い記録、重複した記録、もう不要な記録を特定
3. 整理案を提示 → 承認後に実行

## 記録しないもの

```yaml
不要:
  - 単純なファイル編集・軽微な修正
  - 一般的な知識（AI が元々知っていること）
  - 一回きりの作業で再現性がないもの
```

## 設計原則

```yaml
追加インフラ不要: "MEMORY.md（Claude Code 標準機能）のみ使用"
200行制限: "超えたら --clean で棚卸し"
実用本位: "次の自分が助かる情報だけ残す"
```
